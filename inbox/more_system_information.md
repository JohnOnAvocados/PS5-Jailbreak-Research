# More System Information
## Source URL
https://www.psdevwiki.com/ps5/More_System_Information
## System Layer
System Software / Diagnostics
## Summary
A hidden system information menu accessible via button combo that displays detailed version data from `/priv/etc/index.dat`.
## Key Concepts
- Access path: Settings > System > System Software > Console Information, then button combo L1 + L3 + Triangle (5s), release, then D-Pad Up + Options.
- Data source: `priv/etc/index.dat`
- Displays system software version details including release, build type, revision numbers, SDK version, middleware version, VSH version, and framework version.
## System Role
Provides detailed firmware build information for diagnostic purposes.
