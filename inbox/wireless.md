## Summary
The PS5 wireless communication module houses Bluetooth and Wi-Fi. The module is AK8M19DFR1 (M19DFR1), made by Sony.
## Key Concepts
- **Module**: AK8M19DFR1 / M19DFR1
- **Manufacturer**: Sony
- **Functions**: Bluetooth and Wi-Fi communication
## System Role
Wireless communication module for PS5 console providing Bluetooth and Wi-Fi connectivity.
