## Summary
SIE CXD90060GG APU chip. RDNA2 2.23GHz. Codename: Oberon. Includes GDDR6 memory interface (channels E0, E1, D0, D1), SVI2 power management, and extensive pinout for 1.0V SOC power domains.
## Key Concepts
- **Function**: APU (Accelerated Processing Unit)
- **Codename**: Oberon
- **GPU**: RDNA2 2.23GHz
- **Memory Interface**: GDDR6, 4 channels (E0, E1, D0, D1) with DQ, DBI, EDC, WCK signals
- **Power Domains**: VDDCR_GFX (1.0V_SOC_VGFX), VDDCR_CORE (1.0V_SOC_VCORE)
- **Memory Voltage**: 1.35V_G6_VMEMIO
- **Control Interfaces**: SVI2 (SVD0, SVC0, SVT0) for voltage regulation
- **Large BGA package with hundreds of pins for power delivery and signal routing**
## System Role
Main APU (CPU + GPU) for the PlayStation 5.
