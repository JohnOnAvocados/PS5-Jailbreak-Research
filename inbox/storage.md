hardware
## Summary
The PS5 has an 825 GB PCIe 4.0 SSD soldered to the motherboard with a proprietary custom controller and six flash memory chips. Includes a user-accessible M.2 2282 PCIe 4.0 SSD expansion slot. Supports USB extended storage and USB drives.
## Key Concepts
- 825 GB PCIe 4.0 SSD
- Soldered to motherboard (non-replaceable)
- Proprietary custom SSD controller
- Six NAND flash memory chips surrounding controller
- DDR4 memory chip for SSD controller
- User-accessible M.2 2282 PCIe 4.0 SSD slot
- 5.5 GB/s raw read bandwidth
- USB HDD extended storage support
## System Role
Primary and expandable game/data storage
