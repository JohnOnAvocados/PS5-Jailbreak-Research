# Hypervisor Loader

## Source URL
https://www.psdevwiki.com/ps5/Hypervisor_Loader

## System Layer
Hypervisor

## Summary
Introduced in FW 03.00.00.33 and above. Codename: "HyLonome".

## Key Concepts
- First appeared in FW 3.00
- Acts as an intermediate loader between the Secure Loader and the Hypervisor
- Codename suggests "Hypervisor Loader — No ME" or similar

## System Role
The Hypervisor Loader adds an additional stage to the PS5 boot chain (introduced in FW 3.00). This suggests Sony hardened the boot sequence by adding a verified loader stage between the IPL/Secure Loader and the Hypervisor itself.
