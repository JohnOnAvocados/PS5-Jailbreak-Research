# DualSense DFU Modes
## Source URL
https://www.psdevwiki.com/ps5/DualSense_DFU_Modes
## System Layer
hardware
## Summary
Three DFU modes exist for DualSense firmware flashing: Primary Boot Loader (PBL), Secondary Boot Loader (SBL), and Main Mode.
## Key Concepts
- **DFU Primary Boot Loader Mode**: Used to flash only the secondary boot loader (SBL)
- **DFU Secondary Boot Loader Mode**: Used to flash Main Firmware (Banana), Venom Firmware (Audio), and Onion Firmware (DSP)
- **Main Mode**: Used to flash Venom Firmware (Audio), Onion Firmware (DSP), and Bluetooth Patches
## System Role
Firmware update modes for the DualSense wireless controller.
