# Modal Browser
## Source URL
https://www.psdevwiki.com/ps5/Modal_Browser
## System Layer
System Software / Web
## Summary
The PS5 includes a modal web browser that can be called from applications to load webpages, used for the Users Guide, social media login (Twitter/YouTube/Twitch), and similar embedded web functions.
## Key Concepts
- No standalone browser app exists; browser is modal-only (called by other apps).
- Similar implementation to PS3, PS Vita, and PS4 modal browsers.
- Contains an HTTPS bypass vulnerability.
## System Role
Embedded web rendering capability for system-integrated web content and authentication flows.
