# System Software Installation
## Source URL
https://www.psdevwiki.com/ps5/System_Software_Installation
## System Layer
System Software / Recovery
## Summary
Procedures for installing or reinstalling PS5 system software via Safe Mode using a USB drive with a PS5UPDATE.PUP file.
## Key Concepts
- **Partial reinstallation (no data loss)**: Uses SYSTEM PUP file. USB path: `PS5 > UPDATE > PS5UPDATE.PUP`. Select Safe Mode option 3 > 1.
- **Complete reinstallation (data loss)**: Uses RECOVERY PUP file. Same USB structure. Select Safe Mode option 7.
- USB must be formatted FAT32 or exFAT.
- PUP filename must be uppercase `PS5UPDATE.PUP`.
- Cannot install a lower version than the currently installed firmware.
## System Role
Official recovery and update installation procedures for PS5 consoles.
