## Summary
The PS5 uses a custom x86-64 AMD Ryzen-based "Zen 2" CPU with 8 cores and 16 threads, clocked at a variable frequency up to 3.5 GHz.
## Key Concepts
- 8 cores, 16 threads
- x86-64 AMD Ryzen Zen 2 microarchitecture
- Variable frequency up to 3.5 GHz
- Custom semi-custom AMD APU design
## System Role
Primary compute processor
