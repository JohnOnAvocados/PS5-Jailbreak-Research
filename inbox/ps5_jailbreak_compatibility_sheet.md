## Summary
Community-maintained Google Sheets spreadsheet tracking PS5 jailbreak status across all firmware versions. Provides a support matrix mapping userland entry points (WebKit, BD-JB, LUA, Y2JB, NFNH) and kernel exploits to specific firmware ranges. Also tracks PS4/PS5 game backup compatibility, homebrew application support, and payload loader availability per firmware.

## Key Concepts
- Firmware compatibility matrix: which entry points work on each specific fw version
- Userland entry point support: WebKit (1.00-5.50), BD-JB (1.00-7.61), LUA (2.00-LATEST), Y2JB (4.03-13.40), NFNH (4.03-12.XX)
- Kernel exploit coverage: UMTX2, Lapse, Poopsploit, P2JB with version ranges
- PS4/PS5 backup loading status: which fw versions can load dumped games
- Homebrew app compatibility: ItemzFlow, PS5 Debug NG, payload managers
- Payload support: etaHEN, Kstuff Lite availability per firmware
- Updated collaboratively by the PS5 jailbreak community with version tracking
- Useful for determining optimal firmware to stay on for maximum capability

## Security Relevance
Illustrates the fragmented security landscape across PS5 firmware versions. Each firmware patch closes specific exploit classes (kernel, hypervisor, userland) while leaving others open. The matrix maps Sony's evolving mitigation strategy against community-discovered bypasses.
