## Summary
NIST Special Publication 800-155 (Draft) provided guidelines for BIOS integrity measurement and attestation mechanisms to ensure platform boot integrity. It defined requirements for measuring boot firmware components, storing measurements in TPM Platform Configuration Registers (PCRs), and reporting them to remote verifiers. The publication aimed to establish standards for detecting unauthorized modifications to platform firmware at boot time.

## Key Concepts
- BIOS integrity measurement
- Platform boot attestation
- TPM PCR usage for firmware measurement storage
- Remote verification of boot chain
- Measured boot architecture
- Root of trust for measurement and reporting
- Detection of unauthorized firmware modifications

## Security Relevance
Defines the foundational integrity measurement and reporting framework for platform firmware at boot time. Establishes how a trusted platform module (TPM) serves as the root of trust for storing and attesting firmware integrity measurements, which is essential for establishing a chain of trust from the earliest boot stage.
