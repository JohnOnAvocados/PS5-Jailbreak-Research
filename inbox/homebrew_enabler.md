## Summary
Catalog of PS5 homebrew enablers organized by exploit type. HEN via hypervisor exploit (Prosperous HEN FW 1.00-4.51, Byepervisor HEN FW 1.00-2.70), via kernel exploit (HEN-V FW 3.00-4.51, etaHEN FW 2.00-10.01), via JIT code execution (Mast1c0re part 2 FW 2.00-13.00), and partial HEN solutions (kstuff, libhijacker, BackPork, PS4 EBOOT DLC Patcher). PS5 has both kernel and hypervisor securities making HEN harder and less powerful than PS4.
## Key Concepts
- Prosperous HEN: chains TMR vulnerability (fail0verflow) with Byepervisor HEN, FW 1.00-4.51
- Byepervisor HEN: FW 1.00-2.70, via Specter's hypervisor exploit
- HEN-V: kernel-only HEN, FW 3.00-4.51 (astrelsky)
- etaHEN: kernel-only HEN, FW 2.00-10.01 (LightningMods)
- Mast1c0re part 2: JIT execution via PS2 Classics emulator, FW 2.00-13.00, requires Star Wars: Racer Revenge
- kstuff: payload for kernel ELF loading, FW 3.00-12.00
- kstuff-toggle: disables kstuff after game launch for performance
- libhijacker: library injection, FW 3.00-4.51
- BackPork: background auto-sideload payload
- PS4 EBOOT DLC Patcher: PC tool for DLC unlocking
