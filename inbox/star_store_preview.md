# ★ Store Preview

## Source URL
https://www.psdevwiki.com/ps5/%E2%98%85_Store_Preview

## System Layer
System Software

## Summary
Store Preview provides previews for In-Game/Global catalogs of the regional PlayStation Store. Only available on DevKit and TestKit consoles.

## Key Concepts
- DevKit/TestKit only (non-retail)
- On retail machines, requires internet connection to use
- Allows developers to preview store catalog layouts

## System Role
Developer tool for previewing store catalog content presentation during development.
