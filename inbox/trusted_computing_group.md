## Summary
Industry standards body that develops open specifications for trusted computing technologies. TCG created the Trusted Platform Module (TPM) specification, the TCG Software Stack (TSS), and standards for remote attestation, measured boot, and device identity. Maintains specifications for TPM 2.0, trusted network communications, and storage security.

## Key Concepts
- Trusted Platform Module (TPM) 2.0 specification for hardware root of trust
- Measured boot and secure boot chain integrity verification
- Remote attestation protocols for verifying system state
- TCG Software Stack (TSS) API for interacting with TPM hardware
- Trusted Network Communications (TNC) for network access control
- Platform Certificate Management for device identity attestation

## Security Relevance
Defines hardware-rooted trust primitives foundational to trusted systems. TPM provides the lowest-level hardware trust anchor used by operating systems, hypervisors, and security software to establish chain of trust from firmware through bootloader to kernel.
