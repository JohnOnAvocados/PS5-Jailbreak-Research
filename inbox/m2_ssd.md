# M.2 SSD
## Source URL
https://www.psdevwiki.com/ps5/M.2_SSD
## System Layer
hardware
## Summary
M.2 SSD expansion storage for PS5. Requires PCI-Express Gen4x4 NVMe SSD (Key M), 250 GB - 8 TB capacity, sequential read speed 5,500MB/s or faster recommended. Requires heatsink.
## Key Concepts
- **Interface**: PCI-Express Gen4x4 supported M.2 NVMe SSD (Key M)
- **Storage Space**: 250 GB - 8 TB (4 TB before system software 8.00)
- **Form Factors**: 2230, 2242, 2260, 2280, 22110
- **Size with Heatsink**: Width up to 25 mm, Length 30/40/60/80/110 mm, Thickness up to 11.25 mm
- **Heat Dissipation**: Required - heatsink max height 2.45 mm (below), 8 mm (above)
- **Socket Type**: Socket 3 (Key M)
- **System Software**: Requires 4.00 or later; 8.00 enables up to 8 TB
- **Compatibility**: PCIe 3.0 drives can work but require workaround; M.2 SATA SSDs not supported
## System Role
Expandable internal storage for PS5 console.
