## Summary
The ACM Digital Library is the primary repository for Association for Computing Machinery publications, including journals, conference proceedings, magazines, and books. It contains research relevant to boot chain security across ACM conferences including CCS (Computer and Communications Security), CODASPY (Data and Application Security and Privacy), and WiSec (Security and Privacy in Wireless and Mobile Networks). Recent proceedings include security and privacy in cyber-physical systems, secure boot for embedded systems, firmware security analysis, and platform integrity verification. ACM has transitioned to an Open Access model.
