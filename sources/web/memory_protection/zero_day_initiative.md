## Summary
The Zero Day Initiative (ZDI) is Trend Micro's vulnerability acquisition and disclosure program. It is the world's largest vendor-agnostic bug bounty program, paying researchers for responsibly disclosing zero-day vulnerabilities. ZDI publishes advisories for all acquired vulnerabilities after vendor patches are released, including CVEs, CVSS scores, and technical details. The program also hosts the Pwn2Own hacking contest.

## Key Concepts
- largest vendor-agnostic vulnerability acquisition program
- coordinates disclosure with vendors before public release
- publishes advisories with CVE, CVSS, and technical descriptions
- hosts Pwn2Own contest targeting major OS, browser, and hypervisor platforms
- over 19,000 independent external vulnerability researchers in the community
- more than 450 dedicated Trend Micro researchers across 14 threat centers
- proactive filters deployed before vendor patches are available (90+ days average protection)
- supports IPS, endpoint, and cloud workload defenses

## Security Relevance
ZDI advisories document real-world exploited or exploitable vulnerabilities, many of which involve memory corruption (heap overflow, use-after-free, type confusion, out-of-bounds) affecting major operating systems, hypervisors, and applications. These disclosures provide insight into how memory protections are bypassed in practice.
