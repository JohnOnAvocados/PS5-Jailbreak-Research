## Summary
The Stanford Applied Cryptography Group, led by Prof. Dan Boneh, conducts research on cryptographic applications to real-world security problems. Key research areas include lattice-based cryptography, privacy-preserving protocols (Prio, Riposte), pairing-based cryptography, blockchain technologies (Stanford Blockchain Research Center), web security (SSL/TLS), and post-quantum cryptography. The group has produced foundational work in identity-based encryption, short signatures, and zero-knowledge proofs (Bulletproofs).

## Key Concepts
- Lattice-based cryptography: hardness from LWE/SIS problems
- Pairings: identity-based encryption, short signatures
- Bulletproofs: short zero-knowledge proofs for confidential transactions
- Prio: private aggregation of statistics via secret sharing
- Riposte: anonymous messaging system with differential privacy
- Balloon Hashing: memory-hard password hashing resistant to GPU/ASIC
- PowerSpy: side-channel location tracking from power usage
- IBE secure email system
- CS251: Stanford Blockchain course

## Security Relevance
While not directly focused on CPU architecture, the Applied Cryptography Group's work on cryptographic primitives forms the mathematical foundation for trusted system models including attested boot, memory encryption, remote attestation, and secure enclave protocols. Lattice-based cryptography is central to post-quantum secure architectures for trusted platforms. Bulletproofs and related ZK proofs enable verifiable trust properties for confidential computing.
