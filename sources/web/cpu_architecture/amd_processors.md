## Summary
AMD's processor product line spans consumer (Ryzen desktop and mobile), professional (Ryzen PRO, Threadripper), server (Epyc), and embedded (Ryzen Embedded) segments. Key architectures include Zen family (Zen 1 through Zen 5), with process nodes from 14nm to 3nm. Server processors feature up to 128 cores (Epyc Bergamo), PCIe 5.0, DDR5, and Infinity Fabric interconnect. Consumer processors include 3D V-Cache variants and integrated RDNA graphics in APUs.

## Key Concepts
- Ryzen: desktop and mobile consumer processors (Zen 1–Zen 5)
- Ryzen PRO: enterprise security features including hardware root of trust
- Threadripper: high-end desktop (HEDT) with up to 64 cores
- Epyc: server/cloud with up to 128 cores (Genoa, Bergamo, Siena, Turin)
- Ryzen Embedded: long-lifecycle, industrial temperature range
- AM5 socket: LGA1718 supporting DDR5 and PCIe 5.0
- Zen 4c/Zen 5c: high-density cloud-optimized cores
- 3D V-Cache: vertical L3 cache stacking for gaming/server

## Security Relevance
All AMD processors include the AMD Platform Security Processor (PSP), which forms the hardware root of trust. Server Epyc processors additionally support SEV/SEV-ES/SEV-SNP for confidential cloud computing. The security properties of each processor tier directly determine the trusted system model for that platform.
