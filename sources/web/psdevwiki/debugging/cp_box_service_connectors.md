# CP Box Service Connectors
## Source URL
https://www.psdevwiki.com/ps5/CP_Box_Service_Connectors
## System Layer
debugging
## Summary
CP Box provides UART pins for accessing EMC and EAP serial consoles during development and debugging.
## Key Concepts
- UART access to EMC (Embedded Micro Controller)
- UART access to EAP (Embedded Application Processor)
- Physical pin connections on CP Box hardware
## System Role
Debug serial console access points
