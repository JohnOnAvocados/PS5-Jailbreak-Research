# UART
## Source URL
https://www.psdevwiki.com/ps5/UART
## System Layer
Hardware / Debug Interface
## Summary
UART (Universal Asynchronous Receiver/Transmitter) serial debug interface on PS5 motherboards. See Service Connectors page for pinout schemes.
## Key Concepts
- Provides serial debug access to PS5 hardware
- Pinout schemes documented on Service Connectors page
## System Role
Hardware-level serial debug interface for low-level system debugging and diagnosis during development and manufacturing.
