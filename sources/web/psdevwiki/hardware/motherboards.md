## Summary
PS5 motherboard revisions: EDM-010, EDM-020, EDM-030, EDM-040, EDM-041, EDM-044, EDM-AR10, EDM-AK30, EDM-BK21. PS5 Pro (Viola) motherboard revision: VSM-010.
## Key Concepts
- EDM-010: initial retail motherboard revision
- EDM-020, EDM-030, EDM-040, EDM-041, EDM-044: later retail revisions
- EDM-AR10, EDM-AK30, EDM-BK21: additional board variants
- VSM-010: PS5 Pro motherboard model
