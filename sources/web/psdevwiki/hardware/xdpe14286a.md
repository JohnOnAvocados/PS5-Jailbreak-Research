## Summary
Infineon XDPE14286A VRM controller for PS5 APU power regulation. Manages VDDCR_GFX (GPU) and VDDCR_CORE (CPU) voltage rails via SVI2 interface.
## Key Concepts
- **Function**: VRM (Voltage Regulator Module) controller for PS5 APU
- **Interfaces**: SVI2 (SVC/SVD/SVT), I2C (SDA/SCL)
- **Power Rails**: VDDCR_GFX (GPU), VDDCR_CORE (CPU), 5V_MAIN inputs
- **Input Voltages**: 1.8V_SOC_VDDA, 3.3V_VRM, 1.2V_VRM, 12V_MAIN
- **Features**: 8+2 phase PWM control, temperature sensing (TSEN1/TSEN2), power good/fail outputs
- **Key Signals**: PSW_MSOC_PGC (enable), SOC_PWROK, /SOC_VRM_HOT, /SOC_GPU_PCC
## System Role
APU voltage regulator controller for PlayStation 5.
