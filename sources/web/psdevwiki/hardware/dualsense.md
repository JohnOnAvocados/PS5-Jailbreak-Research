## Summary
The DualSense wireless controller for PS5 features haptic feedback, adaptive triggers, built-in microphone, USB Type-C, Bluetooth 2.402-2.48 GHz, and a 1,560 mAh battery. Codename: Bond.
## Key Concepts
- **Codenames**: Bond (DualSense), Spider (Main MCU/DSP), Venom (Audio Codec), Onion (Sub DSP), Walther (Triggers Assembly)
- **Main MCU/DSP**: SIE CXD9006GG (rebranded MediaTek MT3616XXX, ARM Cortex M4 & N9 DSP Core)
- **PMIC**: Dialog DA9087
- **Audio Codec**: Realtek ALC5524 (Venom)
- **Audio Amplifier**: Realtek ALC1016 or Nuvoton NAU8225 3.0W Class-D
- **Main Firmware**: Runs on ARM Cortex M4
- **Revision IDs**: 0x0CE6 (ModelType.BOND), 0x0CE7 (ModelType.ASTON), 0x0D5A (ModelType.LOTUS)
- **USB VID/PID**: 054c:0ce6 Sony Corp.
- **Wireless Interface**: Bluetooth 2.402-2.48 GHz, 2.5 mW power output
- **Wired Interface**: USB Type-C
- **Battery Capacity**: 1,560 mAh
- **USB Classes**: Audio (speaker, microphone), HID
- **Board Revisions**: BDM-010 (retail), BDM-010 R10 (engineering sample with MT3616ECT)
## System Role
Primary input controller for the PlayStation 5 console.
