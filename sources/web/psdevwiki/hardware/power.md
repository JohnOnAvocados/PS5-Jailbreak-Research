# Power
## Source URL
https://www.psdevwiki.com/ps5/Power
## System Layer
hardware
## Summary
The PS5 standard model contains a 350W internal power supply (ADP-400DR); the digital edition uses 340W. Measured peak draw ~207W during gameplay. PSU is multi-voltage (110-240V) across all regions.
## Key Concepts
- 350W PSU (standard model CFI-10XXA)
- 340W PSU (digital edition)
- ADP-400DR internal power supply unit
- Multi-voltage: 110-240V operation
- Peak ~207W during Spider-Man Miles Morales gameplay
## System Role
Power delivery and regulation
