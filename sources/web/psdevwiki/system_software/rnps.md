# RNPS

**Source:** https://www.psdevwiki.com/ps5/RNPS
**System Layer:** System Software
**Summary:** RNPS is the name of the PS5 Home Menu and Background Menu system, encompassing a large suite of applications for UI, settings, social features, media, and system functions.
**Key Concepts:** RNPS, NPXS40xxx, NPXS24xxx, NPXS27xxx, Title IDs, Home Menu, Control Center, shell UI, sandbox mounts
**System Role:** Core user interface framework powering the PS5's home screen, control center, notifications, settings, and all system overlay dialogs.

## Devices

- /dev/rnps/
- /mnt/rnps/ mounted on /dev/lv0/

## Online Connections

RNPS has many dedicated hostnames on playstation.net servers.
