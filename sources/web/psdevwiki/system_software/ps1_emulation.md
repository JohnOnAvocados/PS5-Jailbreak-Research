## Summary
Documentation of PS1 games playable on PS5, including Carbon Engine releases and remasters.

## Key Concepts
- PS1 games on PS5 are primarily delivered through Carbon Engine by Limited Run Games
- Games are released as PS5 native titles with PPSA Title IDs
- PS1 games available via PS4 emulation on PS5 reference the PS4 Dev Wiki

### Notable PS1 Titles on PS5 (Carbon Engine)
- Tomba! Special Edition (PPSA21381, 2024-08-01)
- Clock Tower: Rewind (PPSA21337, 2024-10-29)
- Gex Trilogy (PPSA28542, 2025-06-16)
- Fear Effect (PPSA27321, 2025-08-29)
- Tomba! 2, Fear Effect 2, Fighting Force Collection, Vagrant Story (announced/production)
