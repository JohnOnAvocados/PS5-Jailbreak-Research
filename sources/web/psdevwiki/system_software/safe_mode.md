# Safe Mode
## Source URL
https://www.psdevwiki.com/ps5/Safe_Mode
## System Layer
System Software / Recovery
## Summary
PS5 Safe Mode is a special boot mode for recovery and maintenance tasks, accessed by holding the power button until two beeps. The DualSense controller must be connected via USB.
## Key Concepts
- **Option 1 - Restart PS5**: Ends Safe Mode and reboots normally.
- **Option 2 - Change Video Output**: Change Resolution (1080p/720p) or Change HDCP Mode (enable/disable).
- **Option 3 - Update System Software**: Update from USB Drive (FAT32, PS5 > UPDATE > PS5UPDATE.PUP) or Update Using Internet.
- **Option 4 - Restore Default Settings**: Resets to factory defaults without deleting saved data.
- **Option 5 - Rebuild Database**: Scans SSD and rebuilds content database (fixes freezes, frame rate drops).
- **Option 6 - Reset PS5**: Deletes all user data and restores to original state.
- **Option 7 - Reset PS5 (Reinstall System Software)**: Full recovery via USB recovery PUP, wipes all data.
## System Role
Emergency recovery and maintenance interface for PS5 console.
