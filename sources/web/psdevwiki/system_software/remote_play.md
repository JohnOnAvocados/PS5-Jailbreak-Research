# Remote Play
## Source URL
https://www.psdevwiki.com/ps5/Remote_Play
## System Layer
System Software / Networking
## Summary
PS5 supports Remote Play for remote access from other devices, including official clients for Android, iOS, macOS, PS4, PS5, and Windows, and the open-source Chiaki client for additional platforms.
## Key Concepts
- Official Remote Play supports: Android, iOS, macOS, PS4 (since FW 8.00), PS5, Windows.
- Chiaki (open-source) supports: Android, Linux, macOS, Nintendo Switch, PS5, PS Vita (ported via VitaKi), Windows.
- No official Linux or Nintendo Switch client.
## System Role
Remote streaming functionality allowing PS5 control from external devices over network.
