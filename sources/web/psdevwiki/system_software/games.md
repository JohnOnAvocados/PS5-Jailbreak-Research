# Games

**Source:** https://www.psdevwiki.com/ps5/Games
**System Layer:** System Software
**Summary:** Catalog of PS5 game titles with metadata (genre, developer, publisher, release dates, sizes, Title IDs, firmware requirements).
**Key Concepts:** Title ID, PPSA prefix, firmware requirement, launch games
**System Role:** Reference listing of PS5 game titles and their technical identifiers.

## Game Listings

The wiki page catalogs PS5 games alphabetically with the following fields per entry:
- Title name
- Genre(s)
- Developer(s)
- Publisher(s)
- Release date (Japan, Europe, US)
- Size
- VR Support
- Disc/Digital availability
- Title ID (PPSA format)
- Minimum firmware requirement
- Remark
