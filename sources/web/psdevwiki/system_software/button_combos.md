# Button Combos
## Source URL
https://www.psdevwiki.com/ps5/Button_Combos
## System Layer
System Software / Shell
## Summary
Button combinations that trigger special system menus and debug functions on PS5.
## Key Concepts
- **Safe Mode**: While console is off, hold power button until two beeps.
- **More System Information**: In Settings > System > System Software > Console Information, hold L1 + L3 + Triangle for 5 seconds, then release and press D-Pad Up + Options.
- **Staff Mode (IDU)**: Hold L1 + L2, then press Circle, Cross, Square, Triangle, D-Pad Right in sequence, then release L1 + L2.
- **PsnInGameCommerce**: In debug settings, press D-Pad Left + Square + R1 simultaneously.
- **Toggle Extra Debug Menu**: On main menu, press Start + L3 simultaneously (requires SblRcMgrIsAllowDebugMenuForSettings enabled).
- **Crash The Shell**: With registry option enabled, various 4-button combos involving D-Pad Up + L1 + R1 + another button.
## System Role
Provides hidden diagnostic and debug system access via controller input sequences.
