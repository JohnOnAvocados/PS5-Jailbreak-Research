## Summary
Reference for PS5 demo/trial games available on the PlayStation Store.

## Key Concepts
- PS5 demo games are listed on PlayStation Store under category ID `95601a70-7564-4771-b305-0283fe3593e4`
- URL format: `https://store.playstation.com/<region>/category/95601a70-7564-4771-b305-0283fe3593e4`
- PS4 demo games can usually be played on PS5 via backward compatibility
