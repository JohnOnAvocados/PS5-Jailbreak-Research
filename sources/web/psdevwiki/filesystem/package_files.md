# Package Files
## Source URL
https://www.psdevwiki.com/ps5/Package_Files
## System Layer
filesystem
## Summary
PS5 package file header structure: magic bytes 7F 43 4E 54 (CNT retail) or 7F 46 49 48 (FIH debug). Same magic as PS4. PkgType 83 02 identifies PS5 packages. Revision 00 01. Game folders mounted at /app0/gameName.
## Key Concepts
- Retail magic: 7F 43 4E 54 (?CNT)
- Debug magic: 7F 46 49 48 (?FIH)
- Same magic as PS4
- PkgType: 83 02 (identifies PS5 package)
- Revision: 00 01
- Game mount point: /app0/gameName
