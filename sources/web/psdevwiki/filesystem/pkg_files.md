# PKG Files

**Source:** https://www.psdevwiki.com/ps5/PKG_files
**System Layer:** Filesystem
**Summary:** Documentation and tool references for PS5 PKG file format, decryption, and extraction.
**Key Concepts:** NPDRM, keyset 3, pkgdec5, PS5PKGReader, PFS
**System Role:** Reference page for third-party tools used to analyze and extract PS5 PKG package contents.

## Overview

Documentation about PS5 PKG and PFS is available at the etaHEN project repository: [https://github.com/etaHEN/etaHEN/blob/main/PS5%20technical%20writeups/pkg-writeup.md](https://github.com/etaHEN/etaHEN/blob/main/PS5%20technical%20writeups/pkg-writeup.md)

## Tools

### pkgdec5 by Red-EyeX32 and zecoxao
Open source C# GUI program to decrypt and extract PS5 release or debug NPDRM sc PKG files encrypted with keyset 3.
- Source: [https://github.com/zecoxao/pkgdec5](https://github.com/zecoxao/pkgdec5)

### PS5PKGReader by SvenGDK
Open source C# GUI program to partially extract (without decryption) PS5 release or debug NPDRM PKG files.
- Source: [https://github.com/SvenGDK/PS5PKGReader](https://github.com/SvenGDK/PS5PKGReader)
