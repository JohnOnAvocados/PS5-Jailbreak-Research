# Secure Modules
## Source URL
https://www.psdevwiki.com/ps5/Secure_Modules
## System Layer
security
## Summary
PS5 secure modules (identified by 0x8002xxxx IDs) providing trusted services: authmgr, kms, pup, pfs, driveauth, pltauth, npdrm, devact, qafutkn, sysveri, otpaccess, manu, fttrm, srtc, rootparam, exthdd, cloudsd, bar, otprsvaccess, diskid, idata, ddd, otpctrl, ncdt, hidauth.
## Key Concepts
- 0x80021000: authmgr (Authentication Manager)
- 0x80021001: kms (Key Management System)
- 0x80021002: pup (PUP firmware update)
- 0x80021003: pfs (PlayStation File System)
- 0x80021004: driveauth (Drive Authentication)
- 0x80021005: pltauth (Platform Authentication)
- 0x80021006: npdrm (Network Product DRM)
- 0x80021007: devact (Device Activation)
- 0x80021008: qafutkn (QA/Utoken)
- 0x80021009: sysveri (System Verification)
- 0x8002100A: otpaccess (OTP Access)
- 0x8002100B: manu (Manufacturing)
- 0x8002100C: fttrm (Film/TV Tracking Rights Management)
- 0x8002100D: srtc (Secure RTC)
- 0x8002100E: rootparam (Root Parameter)
- 0x8002100F: exthdd (External HDD)
- 0x80021010: cloudsd (Cloud SaveData)
- 0x80021011: bar (Backup and Restore)
- 0x80021012: otprsvaccess (OTP Reserved Access)
- 0x80021013: diskid (Disk ID)
- 0x80021014: idata (Identity Data)
- 0x80021015: ddd
- 0x80021016: otpctrl (OTP Control)
- 0x80021017: ncdt
- 0x80021018: hidauth (HID Authentication)
## System Role
Trusted service modules for secure system operations
