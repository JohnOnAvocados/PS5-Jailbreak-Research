# SKU Regioning
## Source URL
https://www.psdevwiki.com/ps5/SKU_Regioning
## System Layer
Hardware / Regional Configuration
## Summary
Table of regional codes used in PS5 SKU naming, mapping region abbreviations to their geographic markets and Target IDs.
## Key Concepts
- J = JAPAN (TARGET ID 0x83)
- AU3 = AUSTRALIA (TARGET ID 0x89)
- RU3 = RUSSIA (TARGET ID 0x8C)
- CN9 = CHINA
- BR2 = BRAZIL
- UCS = UNITED STATES (TARGET ID 0x84)
- EU8 = EUROPE (TARGET ID 0x87)
- E32 = ASIA (TARGET ID 0x8A)
## System Role
Defines the regional naming convention for PS5 SKU models, used to identify target markets and configure region-specific firmware behavior.
