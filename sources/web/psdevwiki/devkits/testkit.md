# TestKit
## Source URL
https://www.psdevwiki.com/ps5/TestKit
## System Layer
Hardware / Testing Hardware
## Summary
The PS5 TestKit is used for testing and quality assurance, with 16GB GDDR6 memory (same as retail but with test-specific firmware capabilities).
## Key Concepts
- 16GB GDDR6 memory (same as retail)
- FCC ID: NCC/CCAB20LP129BT4
- Model: DFI-T1000AA
- Has a final outer shell (unlike DevKit)
## System Role
Testing hardware used for QA and certification testing of PS5 software, with retail-equivalent memory but access to test/debug firmware features.
