# DevKit
## Source URL
https://www.psdevwiki.com/ps5/DevKit
## System Layer
Hardware / Development Hardware
## Summary
The PS5 DevKit has 32GB GDDR6 memory (double the 16GB in TestKit and retail consoles). It is used by developers for game development and testing.
## Key Concepts
- 32GB GDDR6 memory (vs 16GB on TestKit/retail)
- FCC ID: NCC/CCAB20LP1680T0
- Model number: DFI-D1000AA
## System Role
Development hardware platform for creating and debugging PS5 games and applications, with doubled memory capacity compared to retail units.
