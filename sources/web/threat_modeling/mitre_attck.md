## Summary
MITRE ATT&CK is a globally-accessible knowledge base of adversary tactics and techniques based on real-world observations. It covers Enterprise, Mobile, and ICS environments with matrices organized by tactics (e.g., Reconnaissance, Execution, Persistence, Privilege Escalation, Defense Evasion, Credential Access, Discovery, Lateral Movement, Collection, Command and Control, Exfiltration, Impact), each containing multiple techniques and sub-techniques identified by T-codes.

## Key Concepts
- Adversary tactics (15 enterprise tactics) organized in matrices
- Techniques with sub-techniques for granular adversary behavior mapping
- T-code identification system for each technique
- Covers Enterprise, Mobile, and ICS domains
- Mitigations, detections, and data components mapped to techniques
- Groups, software, and campaigns linked to techniques used
- Community-driven, open and free to use
- Foundation for threat model development and methodology

## Security Relevance
Provides the authoritative taxonomy of adversary behaviors that any trusted system threat model must account for. Directly informs the "what can go wrong" dimension of threat modeling by enumerating specific attack vectors across all system layers.
