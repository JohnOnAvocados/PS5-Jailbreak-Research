## Summary
This PhD thesis by Hamed Nemati (2020) explores building a provably secure separation kernel using virtualization technology. It presents end-to-end formal verification of memory isolation in a security kernel, focusing on memory subsystem management. The work addresses techniques for constructing high-assurance mixed-criticality execution platforms where a thin trusted computing base enforces isolation between low- and high-criticality components. Formally verified memory isolation reduces the system attack surface and facilitates security certification.

## Key Concepts
- Separation kernel architecture via virtualization
- Formal verification of memory isolation
- Mixed-criticality execution platforms
- Trusted Computing Base (TCB) minimization
- End-to-end security guarantees
- Memory subsystem management for isolation
- High-assurance kernel design

## Security Relevance
This research directly addresses the problem of proving that a hypervisor-based separation kernel correctly enforces memory isolation. Formal verification of the isolation properties provides the highest assurance level for trusted systems, relevant to certification standards like Common Criteria EAL7 and high-security deployments.
