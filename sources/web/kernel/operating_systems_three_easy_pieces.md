## Summary
OSTEP (Operating Systems: Three Easy Pieces) is a free, open online textbook by Remzi and Andrea Arpaci-Dusseau covering virtualization, concurrency, and persistence. Version 1.10 includes security chapters on authentication, access control, cryptography, and distributed security by Peter Reiher.

## Key Concepts
- Virtualization (CPU, memory, address spaces)
- Concurrency (threads, locks, condition variables, semaphores)
- Persistence (filesystems, storage devices, journaling)
- Security chapters (intro, authentication, access control, cryptography, distributed)
- Process scheduling (MLFQ, lottery scheduling)
- Memory management (paging, TLB, swapping)
- File system implementation (FFS, LFS, journaling)
- xv6 lab projects

## Security Relevance
Standard OS textbook covering foundational operating system concepts necessary for understanding kernel security. Security chapters directly address authentication, capabilities, access control, and cryptography within OS design.
