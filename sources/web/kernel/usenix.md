## Summary
USENIX is a nonprofit organization that hosts premier conferences for systems research including OSDI (Operating Systems Design and Implementation), USENIX Security, and NSDI (Networked Systems Design and Implementation). All conference proceedings are freely accessible with no-fee open access policy.

## Security Relevance
Primary venue for cutting-edge operating systems security research. USENIX Security and OSDI publish fundamental work on kernel exploits, trusted execution, isolation mechanisms, and systems security.
