## Summary
NIST is the US National Metrology Institute that develops measurement standards, cybersecurity frameworks (NIST Cybersecurity Framework, SP 800 series), cryptographic standards, and operates the National Vulnerability Database (NVD). Key for cybersecurity policy and standards.

## Key Concepts
- NIST Cybersecurity Framework (CSF)
- NIST SP 800 series (security and privacy controls)
- National Vulnerability Database (NVD) integration with CVE
- Cryptographic standards (FIPS 140-3, post-quantum cryptography)
- NIST SP 800-53 (security and privacy controls for information systems)
- NIST SP 800-190 (application container security guide)
- Computer Security Resource Center (CSRC)
- AI risk management framework

## Security Relevance
NIST defines standards and guidelines governing secure system configuration, cryptographic validation, and vulnerability management. Essential for compliance-driven kernel hardening and security assessment in a trusted system model.
