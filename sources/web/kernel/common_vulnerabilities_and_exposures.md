## Summary
The CVE (Common Vulnerabilities and Exposures) system, managed by the MITRE Corporation, provides a standardized identifier system for publicly disclosed cybersecurity vulnerabilities. Each CVE ID uniquely identifies a specific vulnerability with a description and references.

## Key Concepts
- CVE ID format (CVE-YYYY-NNNNNN)
- CVE Numbering Authorities (CNAs) assign IDs
- CVE Record contains description, references, and phase
- CVE List is the authoritative dictionary of publicly known vulnerabilities
- CVE to CWE (Common Weakness Enumeration) mapping
- CVSS (Common Vulnerability Scoring System) severity scores
- CVE.org website (mitre managed, JS-dependent)

## Security Relevance
CVE is the global standard for vulnerability identification. Kernel CVEs document all publicly reported kernel security flaws, making it essential for tracking kernel exploits, patching, and vulnerability research in a trusted system model.
