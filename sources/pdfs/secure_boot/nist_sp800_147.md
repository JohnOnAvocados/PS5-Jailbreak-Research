# NIST SP 800-147 — BIOS Protection Guidelines

## Source
https://nvlpubs.nist.gov/nistpubs/Legacy/SP/nistspecialpublication800-147.pdf

## Domain
Secure Boot / Firmware Integrity

## Summary
Defines protection mechanisms for BIOS/UEFI firmware integrity and update security.

## Key Concepts
- Authenticated BIOS updates
- Firmware integrity verification
- Root of trust for firmware update
- Protection against malicious firmware modification
