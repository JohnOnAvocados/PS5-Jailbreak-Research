## Summary
The AMD Secure Processor (formerly Platform Security Processor or PSP) is a dedicated ARM Cortex-A5-based security subsystem integrated into AMD CPUs and APUs. It operates independently of the main x86 cores and runs its own firmware. It handles secure boot verification, cryptographic operations, Trusted Platform Module (fTPM) 2.0 functionality, and memory encryption (SME/SEV).
