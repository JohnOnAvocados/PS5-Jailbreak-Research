## Summary
Intel's security portal outlines hardware-enabled security features including Intel Boot Guard, Intel Trusted Execution Technology (TXT), Intel Software Guard Extensions (SGX), Intel Converged Security and Management Engine (CSME), and Platform Firmware Resilience (PFR). Intel emphasizes security built into silicon as the root of trust, with capabilities spanning endpoint security, network security, data security (confidential computing), and physical security. The portal includes the Product Security Center for advisories, the Intel Security ebook, the annual Platform Security Report, and developer guidance on software security.
