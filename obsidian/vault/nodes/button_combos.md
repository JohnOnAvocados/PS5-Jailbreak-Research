## Summary
Button combinations that trigger special system menus and debug functions on PS5. ## Key Concepts
- **Safe Mode**: While console is off, hold power button until two beeps. - **More System Information**: In Settings > System > System Software > Console Information, hold L1 + L3 + Triangle for 5 seconds, then release and press D-Pad Up + Options.
