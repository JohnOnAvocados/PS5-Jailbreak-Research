## Summary
The PS5 has an 825 GB PCIe 4.0 SSD soldered to the motherboard with a proprietary custom controller and six flash memory chips. Includes a user-accessible M.2 2282 PCIe 4.0 SSD expansion slot. Supports USB extended storage and USB drives.
