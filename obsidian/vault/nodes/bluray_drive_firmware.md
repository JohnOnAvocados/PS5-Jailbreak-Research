## Summary
PS5 Blu-ray drive firmware versions for the 502R drive model, mapped to system software versions. ## Key Concepts
- **Drive Model**: 502R
- **Firmware Versions**:
 - BD 1072: System Software 1.00
 - BD 1073: System Software 1.05
 - BD 1119: System Software 3.00
 - BD 1137: System Software 4.03
 - BD 1275: System Software 8.60
## System Role
Optical disc drive firmware for PS5 Blu-ray drive.
