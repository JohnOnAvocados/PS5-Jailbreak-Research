## Summary
Official Sony publishing tools for PS5 game/application package creation and content management. ## Key Concepts

### Official Tools
- **prospero-pub-param.exe** - Param File Editor for PS5 (creates/verifies .json param files for apps, patches, DLC)
- **prospero-pub-cmd.exe** - Publishing Tools CLI (CUI versions of features including Package Generator)
- **vagconv2w.exe / vagconv2.exe** - VAG Converter 2 (converts 16-bit linear PCM to HE-VAG data)
- **at9tool.exe** - ATRAC9 Encoder/Decoder
- **ProsperoGP5Editor.exe** - GP5 Generator for PS5 (scans game folder structure and generates .gp5 project files for prospero-pub-cmd.exe)
- **sieaacformattool.exe** - AAC Format Tool (encoding/decoding for AAC)
- **sieOpusFormatTool.exe** - Opus Format Tool (encoding/decoding for Opus)

## System Role
These are the official Sony SDK tools used by licensed developers to create PS5 packages, convert audio assets, and manage game content metadata. Understanding these tools is essential for analyzing the PS5 package format.
