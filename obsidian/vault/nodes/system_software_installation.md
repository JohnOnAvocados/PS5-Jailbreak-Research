## Summary
Procedures for installing or reinstalling PS5 system software via Safe Mode using a USB drive with a PS5UPDATE.PUP file. ## Key Concepts
- **Partial reinstallation (no data loss)**: Uses SYSTEM PUP file. USB path: `PS5 > UPDATE > PS5UPDATE.PUP`.
