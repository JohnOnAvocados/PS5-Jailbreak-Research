## Summary
Trusted Firmware provides open source reference implementations of secure software for Arm architectures. It includes Trusted Firmware-A (TF-A) for Armv8-A/Armv9-A application processors and Trusted Firmware-M (TF-M) for Armv8-M microcontrollers. The project provides SoC developers and OEMs with a reference trusted code base complying with Arm specifications, forming the foundation for Trusted Execution Environments (TEE) on application processors and Secure Processing Environments (SPE) on microcontrollers.
