# IDU Mode

**Source:** https://www.psdevwiki.com/ps5/IDU_Mode
**System Layer:** System Software
**Summary:** IDU (Individual Display Unit) mode for kiosk/demo PS5 consoles. **Key Concepts:** IDU Daemon, IDU Client, IDU Utility, Kiosk mode, Demo mode, Title IDs, sflash/NOR flag, PSN restriction
**System Role:** Specialized system mode for retail demo/kiosk consoles that restricts functionality and manages demo content delivery. ## Overview

IDU (Individual Display Unit) is a mode designed for Kiosk/Demo PS5 consoles meant to demonstrate different game demos, features, etc., usually found in game/electronic stores.
