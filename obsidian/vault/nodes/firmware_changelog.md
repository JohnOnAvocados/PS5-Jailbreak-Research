## Summary
Changelog for PS5 system software updates, covering versions from 20.02-02.25.00 (Nov 2020) through 21.01-05.10.00 (May 2022). ## Key Concepts

### Notable Changes
- **02.25.00**: Initial post-launch update; improved system performance, fixed download queue issue
- **02.26.00**: Fixed disc game deletion bug, fixed DualSense charging via front USB-A in rest mode
- **02.30.00**: Fixed data transfer/download cancellation, PS4 text input errors, Wi-Fi stability
- **02.50.00**: Fixed PS4 disc auto-install after PS5 upgrade; added Share Factory Studio clip editing
- **03.00.00**: Full changelog available externally
- **03.20.00**: DualSense firmware update, screen reader fixes, hidden game fix, USB storage copy stability
- **04.00.00**, **05.00.00**: Full changelogs available externally

## System Role
Provides a detailed history of PS5 firmware changes for tracking when specific features and bug fixes were introduced.
