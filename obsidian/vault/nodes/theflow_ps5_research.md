## Summary
TheFloW (Andy Nguyen) is the most prolific PS5 security researcher, responsible for multiple critical vulnerabilities and exploits. BD-JB (Blu-ray Disc Java Sandbox Escape) chains 5 vulnerabilities in the BD-J stack for firmware-agnostic, ROP-less code execution — presented at HardWear.io 2022 and Hexacon 2022, extended to 7.61 via path traversal. Discovered CVE-2020-7457 (IPv6_2292PKTOPTIONS UAF), a kernel exploit via setsockopt race condition originally on PS4 and ported to PS5.
