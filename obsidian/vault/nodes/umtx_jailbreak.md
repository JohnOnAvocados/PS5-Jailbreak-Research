## Summary
UMTX Jailbreak by idlesauce (based on work by shahrilnet, n0llptr, SpecterDev, ChendoChap) is a WebKit-based kernel exploit and jailbreak for PS5 supporting FW 1.00-5.50. Uses PSFree 150b by abc as the WebKit userland entry point. Exploits a use-after-free vulnerability in the FreeBSD umtx (user mutex) subsystem to achieve kernel code execution.
