## Summary
Y2JB is a YouTube-based userland exploit framework for PS5 developed by Gezine. Exploits the PS5 YouTube TV application to execute arbitrary JavaScript via the V8 engine on FW 4.03-13.40. When paired with the P2JB kernel exploit, achieves full jailbreak on FW 9.00-12.70.
