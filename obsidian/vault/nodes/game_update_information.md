# Game Update Information

**Source:** https://www.psdevwiki.com/ps5/Game_Update_Information
**System Layer:** System Software
**Summary:** Catalog of PS5 game update metadata (disk sizes, Title IDs, firmware versions). **Key Concepts:** Title ID, PPSA prefix, disk size, firmware requirement
**System Role:** Reference listing of PS5 game update information for patch and compatibility tracking. ## Game Update Listings

The wiki page catalogs PS5 game updates alphabetically with the following fields per entry:
- Title name
- Developer/Publisher
- Release date
- Pictures
- Videos
- Title ID (PPSA format)
- Disk size (BD/NP)
- Minimum firmware requirement
- Remark
