## Summary
PS5 serial number decoding guide for determining factory firmware version, model (FAT/Slim/Pro), manufacturing date, and region across CFI-10 through CFI-71 models from 2020 to 2026. ## Key Concepts
- **Serial Format**: S01-[Region Letter][Model Digit][Year Digit][Month/Week Letter]xxxxxxxxxxxx
- **Chassis Types**: FAT (CFI-10/11/12, 825GB), Slim (CFI-20/21, 848GB/1TB), Pro (CFI-70/71, 2TB)
- **FAT Firmware Range**: 1.00 (CFI-10) through 7.61 (CFI-12)
- **Slim Firmware Range**: 7.00 (CFI-20) through 13.20 (CFI-21)
- **Pro Firmware Range**: 9.05 (CFI-70) through 13.20 (CFI-71)
- **Region Codes**: AJ/AK (USA), S01 (UK/EU), J (Japan), various others
- **Fan Models**: NMB, Delta, Nidec, Foxconn variants per chassis type
- **Bundles**: Specific game bundles correlate to firmware ranges
## System Role
Serial number decoding reference for identifying PS5 factory firmware and specifications.
