## Summary
The CP Box (model CPBH-100) is a test/debug device connecting to a PS5 TestKit via USB-C. It has Engineering and Normal modes, provides Ethernet connectivity (DEV LAN), and controls debug mode. Without a CP Box, TestKits boot in Release Mode.
