## Summary
Developer tool on DevKit/TestKit for previewing how game hubs will appear to end users. Supports testing with Concept IDs, Title IDs, or Product IDs across three lifecycle states. ## Key Concepts
- Only available on DevKit/TestKit consoles
- Input types: Concept ID, (NP) Title ID, or Product ID
- Three lifecycle preview modes:
 - **Concept Announced** (Coming Soon) — no product available yet
 - **Product Available** — one or more products for sale
 - **Product Sold** — user has purchased the game
- Retail use requires network connection

## System Role
Developer UI preview tool for the PS5 game hub experience, allowing testing across different product lifecycle stages.
