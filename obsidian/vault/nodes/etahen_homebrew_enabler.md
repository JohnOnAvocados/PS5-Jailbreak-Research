## Summary
etaHEN by LightningMods is the primary all-in-one homebrew enabler for PS5, supporting FW 3.00-10.01 (latest version 2.4B). Replaces Debug Settings with a custom Toolbox UI and provides Rest Mode, Remote Play, Plugin/Payload ELF Menu, and External HDD Menu. Integrates Kstuff for fself/fpkg support, ItemzFlow for backup management, a game overlay (CPU/GPU temp, utilization, local IP), Direct PKG Installer v2 with WebUI on port 12800, FTP server on port 1337, Klog server on 9081, and ELF loader on port 9021.
