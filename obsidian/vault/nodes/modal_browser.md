## Summary
The PS5 includes a modal web browser that can be called from applications to load webpages, used for the Users Guide, social media login (Twitter/YouTube/Twitch), and similar embedded web functions. ## Key Concepts
- No standalone browser app exists; browser is modal-only (called by other apps). - Similar implementation to PS3, PS Vita, and PS4 modal browsers.
