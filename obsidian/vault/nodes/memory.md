## Summary
The PS5 contains 16 GB of GDDR6 memory using eight 2 GB modules manufactured by Micron Technology. Part number MT61K512M32KPA-14C:B (FBGA code D9XKV). ## Key Concepts
- 16 GB GDDR6 total
- Eight 2 GB modules
- Micron Technology manufacturer
- Part number: MT61K512M32KPA-14C:B
- FBGA code: D9XKV
- 256-bit memory bus
- 448 GB/s bandwidth
## System Role
Main system RAM (unified memory pool)
