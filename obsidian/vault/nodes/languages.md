## System Layer
System Software / Localization
## Summary
Catalog of supported system languages on PS5 (30 languages total), with letter codes, localized names, and firmware version support across PlayStation platforms. ## Key Concepts
- 30 supported system languages numbered 00-30
- Each language has: number, still image (icon0_XX.png), update info (changeinfo_XX.xml), letter code, localized name, and English name
- Still image format: 512x512px 24bit PNG noninterlaced
- Change info format: UTF-8, maximum 64KB
- Speech recognition supported for: Japanese, English (US), French (France), Spanish (Spain), German, Italian, Spanish (Latin America)
- Unreleased languages (31-34): Hebrew, Hindi, Malay, Slovak
- Language 30 (Ukrainian) added in PS5 FW 3.00+
## System Role
System localization and language selection infrastructure for PS5 UI and store content.
