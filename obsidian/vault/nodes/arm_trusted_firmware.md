## Summary
Trusted Firmware-A (TF-A) is a reference implementation of secure world software for Arm A-Profile architectures (Armv8-A and Armv7-A). It includes an Exception Level 3 (EL3) Secure Monitor and provides a starting point for productization of secure world boot and runtime firmware in AArch32 or AArch64 execution states. TF-A implements Arm interface standards including PSCI, TBBR-CLIENT, SMC Calling Convention, SCMI, and SDEI.
