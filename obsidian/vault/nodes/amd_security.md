## Summary
AMD's security portal provides information on platform security features including AMD Secure Processor (AMD-SP), Secure Encrypted Virtualization (SEV), Secure Memory Encryption (SME), and Platform Secure Boot. The AMD Secure Processor is a dedicated ARM Cortex-A5-based security processor that manages boot, encryption keys, and attestation. AMD SEV provides encrypted VM memory for confidential computing.
