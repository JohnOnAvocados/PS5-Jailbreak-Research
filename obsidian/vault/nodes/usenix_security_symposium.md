## Summary
The USENIX Security Symposium is a premier academic security conference covering a broad range of computer security topics. The conference publishes peer-reviewed research on boot chain security, trusted execution environments, firmware attacks, secure boot vulnerabilities, hardware security modules, and operating system security. Relevant past publications have covered TPM attacks (e.g., CVE-2018-6622 presented at USENIX Security), Trusted Boot vulnerabilities, UEFI firmware attacks, and secure boot bypass techniques.
